# flutter_wechat

#### 🔥A flutter app which clones wechat，it's on the roadmw！ 这是一个仿照微信样式基于flutter实现的app，还在继续完善中，欢迎提pull request 

> 包含了flutter中基本所有的组件，适合以此项目进行flutter学习。
